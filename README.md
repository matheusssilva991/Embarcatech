# Embarcatech - Projetos de Sistemas Embarcados

Este repositório reúne os projetos, exercícios e entregas desenvolvidos durante a residência em sistemas embarcados **Embarcatech**.

## Visão Geral

O conteúdo está organizado em duas fases principais:

- [Fase 1](fase_1/README.md): fundamentos do RP2040, GPIO, interrupções, temporizadores, PWM, ADC e projeto final.
- [Fase 2](fase_2/README.md): atividades avançadas com multitarefa, comunicação, sensores, conectividade, IA e aplicações integradas.

Cada pasta de projeto contém os arquivos necessários para compilação com o Raspberry Pi Pico SDK e, quando aplicável, a configuração para simulação no Wokwi.

## Tecnologias e Ferramentas

- **Placa de desenvolvimento:** BitDogLab com Raspberry Pi Pico W (RP2040)
- **Linguagem de programação:** C com o Raspberry Pi Pico SDK
- **Ambiente de desenvolvimento:** VS Code, CMake, SDK oficial da Raspberry Pi
- **Simulação:** Wokwi em projetos compatíveis

## Estrutura

- `fase_1/`: primeiros exercícios e consolidação dos recursos básicos da plataforma.
- `fase_2/`: projetos com maior integração entre periféricos, módulos e estratégias de software.

## Objetivo

Documentar a evolução dos estudos e práticas realizadas ao longo da residência, com foco em aplicações reais de sistemas embarcados utilizando o microcontrolador RP2040.
