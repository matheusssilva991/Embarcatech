# Embarcatech - Projetos de Sistemas Embarcados

Este repositório reúne os projetos e atividades desenvolvidos durante a residência em sistemas embarcados **Embarcatech**.

## 🔧 Tecnologias e Ferramentas

- **Placa de desenvolvimento:** BitDogLab com Raspberry Pi Pico W (RP2040)
- **Linguagem de programação:** C com o Raspberry Pi Pico SDK
- **Ambiente de desenvolvimento:** VS Code, CMake, SDK oficial da Raspberry Pi

## 📌 Objetivo

Documentar e compartilhar a evolução dos estudos e práticas realizadas ao longo da residência, com foco em aplicações reais de sistemas embarcados utilizando o microcontrolador RP2040.
